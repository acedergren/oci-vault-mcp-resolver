OCI Vault Configuration

How to use OCI Vault references

Use the following URL formats in your MCP configuration:

  • oci-vault://ocid1.vaultsecret.oc1.xxx - Direct secret OCID (fastest)
  • oci-vault://ocid1.compartment.oc1..xxx/secret-name - Compartment + name (recommended)
  • oci-vault://ocid1.vault.oc1.xxx/secret-name - Vault + name
